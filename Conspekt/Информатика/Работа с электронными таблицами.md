### Определение
Электронная таблица (ЭТ) - компьютерная программа позволяющая хранит, организовывать, анализировать и представлять данные в виде таблиц состоящих из строк и столбцов.
Сегодня мы будем работать с Google Sheets. Он поддерживает сотни функций для расчетов и анализа данных. Формула начинается со знака = , а функции это встроенные команды такие как (sam) или (if). Полный список функций доступен через меню справок.

- **SUM** - Суммирует числа.
- **AVERAGE** - Вычисляет среднее значение.
- **MIN** - Находит минимальное значение.
- **MAX** - Находит максимальное значение.
- **COUNT** - Считает количество чисел.
- **IF** - Задает условие ("если...то...").
- **AND** - Проверяет, выполняются ли все условия (И).
- **OR** - Проверяет, выполняется ли хотя бы одно условие (ИЛИ).
- **COUNTIF** - Считает ячейки, отвечающие условию.
- **VLOOKUP** - Ищет значение в первом столбце таблицы и возвращает значение из другой колонки.
- **HLOOKUP** - То же, что VLOOKUP, но для горизонтальных таблиц (по строкам).
- **INDEX** - Возвращает значение из определенной позиции в диапазоне.
- **MATCH** - Находит позицию значения в диапазоне.
- **CONCATENATE** - Объединяет текст из нескольких ячеек.
- **LEN** - Считает количество символов в ячейке.
- **DATE** - Создает дату из чисел (год, месяц, день).
- **TODAY** - Возвращает текущую дату.
- **SORT** - Сортирует диапазон по указанным столбцам.
- **FILTER** - Фильтрует диапазон по заданным условиям.
- **IMPORTRANGE** - Импортирует данные из другого файла Google Таблиц.
