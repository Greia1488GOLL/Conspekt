### 1. Two Sum
```python
class Solution:

    def twoSum(self, nums: List[int], target: int) -> List[int]:

  

        first_op = 0

        second_op = 0

  

        for i in range(len(nums)):

            first_op = nums[i]

            for j in range(i+1, len(nums)):

                second_op = nums[j]

                if first_op + second_op == target:

                    return [i, j]
```

Этот код решает задачу поиска двух чисел в массиве, сумма которых равна заданному значению, используя метод полного перебора. Он работает через двойной цикл: внешний цикл последовательно берёт каждый элемент массива, а внутренний цикл перебирает все последующие элементы, проверяя суммы всех возможных пар. Когда находится пара чисел, дающая в сумме целевое значение, программа немедленно возвращает индексы этих элементов. Такой подход гарантированно находит решение, но не является оптимальным по скорости для больших массивов данных, поскольку требует проверки всех возможных комбинаций пар чисел.

1. Внешний цикл (`for i...`) — берёт очередной «якорный» элемент

2. Внутренний цикл (`for j...`) — для этого якоря перебирает все элементы справа от него

3. Проверка (`if first_op + second_op == target`) — сравнивает сумму с искомой
