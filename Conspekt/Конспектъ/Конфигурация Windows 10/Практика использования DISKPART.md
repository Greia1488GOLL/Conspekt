#DiskPart 
#том #Windows #диск #накопитель #настройка #компьютер 

Исходное состояние системы:

- Диск 0: размер 20 ГБ, стиль разделов MBR.
- На диске 0 имеется один первичный раздел размером 10 ГБ с файловой системой NTFS, буква тома C. Остальные 10 ГБ не распределены.
- Других дисков нет.

Ваша задача — выполнить следующие действия с помощью команд DISKPART (последовательно):

1. Просмотреть список дисков.
2. Выбрать диск 0.
3. Полностью очистить диск (удалить все разделы).
4. Преобразовать диск в стиль GPT.
5. Создать раздел EFI (ESP) размером 500 МБ.
6. Назначить созданному разделу букву S.
7. Создать основной раздел, занимающий всё оставшееся пространство.
8. Отформатировать этот раздел в NTFS с меткой "Data".
9. Назначить разделу данных букву D.
10. Просмотреть список разделов на диске 0.
11. Изменить букву раздела данных с D на E.
12. Удалить раздел данных (бывший D, теперь E).
13. Просмотреть итоговую структуру разделов.

### Выполнение 

1. **`list disk`** — показывает список всех физических дисков в системе.
2. **`select disk 0`** — выбирает диск 0 для дальнейшей работы.
3. **`clean`** — полностью удаляет все разделы и разметку с диска (очищает его).
4. **`convert gpt`** — преобразует стиль разделов диска из MBR в GPT.
5. **`create partition efi size=500`** — создает системный раздел EFI размером 500 МБ (нужен для загрузки Windows в режиме UEFI).
6. **`assign letter=S`** — присваивает свежесозданному разделу букву S.
7. **`create partition primary`** — создает основной раздел, занимая всё оставшееся место на диске.
8. **`format fs=ntfs label="Data" quick`** — быстро форматирует раздел в файловую систему NTFS и дает ему имя (метку) "Data".
9. **`assign letter=D`** — присваивает разделу данных букву D.
10. **`list partition`** — показывает список всех разделов на текущем диске (диске 0).
11. **`select partition 2`** — выбирает второй раздел (раздел данных).
12. **`assign letter=E`** — меняет букву выбранного раздела с D на E.
13. **`delete partition`** — удаляет выбранный раздел (в данном случае раздел данных).
14. **`list partition`** — снова показывает список разделов, чтобы увидеть итоговую структуру.
