**Сценарий тренировки DISKPART**

Исходное состояние системы:

- Диск 0: размер 20 ГБ, стиль разделов MBR.
- На диске 0 имеется один первичный раздел размером 10 ГБ с файловой системой NTFS, буква тома C. Остальные 10 ГБ не распределены.
- Других дисков нет.

Ваша задача — выполнить следующие действия с помощью команд DISKPART (последовательно):

1. Просмотреть список дисков.
2. Выбрать диск 0.
3. Полностью очистить диск (удалить все разделы).
4. Преобразовать диск в стиль GPT.
5. Создать раздел EFI (ESP) размером 500 МБ.
6. Назначить созданному разделу букву S.
7. Создать основной раздел, занимающий всё оставшееся пространство.
8. Отформатировать этот раздел в NTFS с меткой "Data".
9. Назначить разделу данных букву D.
10. Просмотреть список разделов на диске 0.
11. Изменить букву раздела данных с D на E.
12. Удалить раздел данных (бывший D, теперь E).
13. Просмотреть итоговую структуру разделов.

### Выполнение 
1. list disk
2. select disk 0
3. clean
4. convert gpt
5. create partition efi size=500
6. assign letter=S
7. create partition primary
8. format fs=ntfs label="Data" quick
9. assign letter=D
10. list partition
11. select partition 2
12. assign letter=E
13. delete partition
14. list partition
