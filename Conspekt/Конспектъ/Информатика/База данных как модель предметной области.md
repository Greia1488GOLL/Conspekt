База данных (БД) - это совокупность данных организованных по определенным правилам, отражающее состояние объекта и их отношений в некоторой предметной области предназначенная для хранения постоянной памяти и ее постоянного применения. 
Предметная область это целевая тема конкретного проекта. В свою очередь доменная или концептуальная модель призвана вызволить значение терминов и понятие их а также установление верных взаимосвязей между различными понятиями

Базу данных можно рассматривать как информационную модель предметной области. 

### Типы баз данных 

- Реляционные базы данных (табличные) - реляционные базы данных хранят информацию в виде **таблиц** со связями между ними.
- Сетевые базы данных - в сетевых базах данных между таблицами и записями может быть несколько различных связей в том числе несколько предков. Особенности сетевой базы в том что в ней запоминаются все связи и все содержимое для каждой из них. Иначе говоря поиск в такой базе индексируется. 
- Иерархические базы данных - это модель организующие данные в древовидную структуру, где каждый дочерний элемент подчиняется только одному родительскому. 

База данных - это структурированное хранилище информации, организованное по определенным правилам, отражающее состояние объектов и их взаимосвязи в конкретной предметной области. Предметная область определяет тему проекта, а концептуальная модель устанавливает значения терминов и связи между понятиями, что позволяет рассматривать БД как информационную модель. Существуют три основных типа БД: реляционные (табличные со связями), сетевые (множественные связи между записями с индексированным поиском) и иерархические (древовидная структура с подчинением дочерних элементов одному родительскому). Выбор типа БД зависит от конкретных задач и требований к структуре данных.
