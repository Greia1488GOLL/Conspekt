#Информатика #Сортировка #Вставка #Выбор #Пузырьковая

**Пузырьковая сортировка** - один из самых простых алгоритмов сортировки. Он основан на последовательном сравнении соседних элементов массива.
Принцип работы алгоритма заключается в том, что элементы массива последовательно сравниваются попарно. Если элементы стоят в неправильном порядке (например, левый больше правого при сортировке по возрастанию), они меняются местами.
Проходы по массиву повторяются до тех пор, пока все элементы не окажутся на своих местах и массив не станет отсортированным.

**Сортировка выбором** - это алгоритм, в котором на каждом шаге выбирается наименьший элемент из неотсортированной части массива. 
Сначала весь массив считается неотсортированным. Алгоритм проходит по всем элементам и находит минимальный элемент. Затем этот элемент меняется местами с первым элементом массива.
Процесс повторяется до тех пор, пока весь массив не будет отсортирован.

**Сортировка вставками** - алгоритм, который работает по принципу постепенной вставки элементов в правильное место. 
Сначала первый элемент считается отсортированным. Затем берётся следующий элемент и сравнивается с элементами в отсортированной части. Если он меньше, элементы сдвигаются вправо, пока не будет найдено правильное место для вставки. После этого элемент вставляется на своё место. Затем алгоритм берёт следующий элемент и повторяет тот же процесс.
