Sqrt(x)
Тут работает метод деления пополам. Берём диапазон от нуля до самого числа, находим середину и смотрим: если её квадрат слишком большой - отбрасываем правую половину, если слишком маленький - левую. Повторяем, пока не найдём нужное целое число.

```typescript
function mySqrt(x: number): number {

    if (x < 2) return x;

    let left = 0;

    let right = x;

    while (left <= right) {

        const mid = Math.floor((left + right) / 2);

        const square = mid * mid;

        if (square === x) {

            return mid;

        } else if (square < x) {

            left = mid + 1;

        } else {

            right = mid - 1;

        }

    }

    return right;

}

  

function mySqrt2(x: number): number {

    if (x < 2) return x;

    let left = 1;

    let right = Math.floor(x / 2);

    while (left <= right) {

        const mid = Math.floor((left + right) / 2);

        const square = mid * mid;

        if (square === x) {

            return mid;

        } else if (square < x) {

            left = mid + 1;

        } else {

            right = mid - 1;

        }

    }

    return right;

}
```

Climbing Strairs
Представь, что ты стоишь перед лестницей. Чтобы попасть на любую ступеньку, ты можешь либо шагнуть с предыдущей, либо перепрыгнуть со ступеньки, которая на две ниже. Получается, что число способов попасть на текущую ступеньку это просто сумма способов попасть на две предыдущие.

Например, на первую ступеньку есть один способ (просто шагнуть), на вторую - два (два шага или один прыжок). Дальше просто складываешь: 1, 2, 3, 5, 8, 13... и так до нужной цифры

```typescript 
function climbStairs(n: number): number {

    if (n <= 2) return n;

    let prev1 = 1;

    let prev2 = 1;

    let current = 0;

    for (let i = 2; i <= n; i++) {

        current = prev1 + prev2;

        prev2 = prev1;

        prev1 = current;

    }

    return current;

}
```
