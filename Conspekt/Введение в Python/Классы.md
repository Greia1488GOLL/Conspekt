Класс - это пользовательский тип данных, конструкция ООП (Объектно-Ориентированного Программирования), которая описывает тот или иной объект, его свойства и способы взаимодействия с ним.

Класс имеет поля - переменные, которые будут принадлежать к его экземпляру, а также методы - функции, которые также можно будет использовать для его экземпляра.

Простыми слова, класс - шаблон для реализации какого-то объекта. Например, мы знаем как должна выглядеть кружка, какие характеристики и действия она имеет. При помощи класса мы описываем "абстрактную маму-кружку", а экземпляры будут являться "настоящими, конкретными" кружками с определёнными характеристиками и действиями "мамы-кружки."

Класс решает несколько важных задач:
- Дублирование кода
- Унификация объектов (одинаковые по задумке объекты действительно будут иметь одинаковые характеристики и функционал)


### Классы в Python

Объявляются при помощи ключевого слова class и указания имени класса. Затем ставится двоеточие и со следующей строки описывается тело класса.
```python
class Python:
	pass
```
Чтобы объявить поля класса используется ключевое слово self, после которого, через точку указывается имя поля. Поле тоже самое что и переменная, но внутри класса. Однако, через self к переменной можно обращаться только из тела метода. В параметрах метода также обязательно должен быть указан self.

```python
class Python:
	def test(self):
		self.msg = "Hello!"
```

Для инициализации полей при создании экземпляра класса используется функция-конструктор. Она обязательно принимает параметр self. Эта функция будет автоматически вызвана при создании экземпляра класса. Если она не объявлена, будет использован пустой конструктор.
```python
class Python:
	#Конструктор!
	def__init__(self):
		self.msg = "Hello"
```


Создание экземпляра класса (объект класса, объект) происходит вне тела класса, при помощи указания имени класса и круглых скобок после имени. Если в конструкторе кроме self указаны другие параметры, то при создании объекта в скобках мы также укажем атрибуты - то, что является параметрами при описании функции.

```python
class Python
	#Конструктор!
	def __init__(self, text):
		self.msg = text
		
python_example = Python("Hello")
```
