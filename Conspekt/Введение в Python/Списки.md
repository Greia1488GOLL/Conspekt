index(item) "Индекс конкретного элемента" Сначала указывается имя списка, через точку вызывается метод index, в которой передается элемент item. В результате работы возвращается индекс первого найденного в списке. Если элемента нет, выбрасывает исключение ValueError

### Len(list) "Длина списка"

Сначала вызывается функция len, а ей в качестве аргумента передаётся конкретный список. Возвращает длину списка в виде целого числа.


###### mi(list) "Минимальное и максимальное значение в списке"

Функции возвращает мин. или макс. значение из указанного в качестве аргумента, списка.

**Sorted** - создает новый отсортированный список: `sorted([3, 1, 2]) → [1, 2, 3]`
**Insert** - вставляет элемент на указанную позицию: `[1, 3].insert(1, 2) → [1, 2, 3]`
**Extend** - добавляет элементы из другого списка: `[1, 2].extend([3, 4]) → [1, 2, 3, 4]`
**Remove** - удаляет первый найденный элемент: `[1, 2, 2].remove(2) → [1, 2]`
**Pop** - удаляет и возвращает элемент по индексу: `[1, 2, 3].pop(1) → возвращает 2, список [1, 3]`
**Count** - подсчитывает количество вхождений: `[1, 2, 1, 1].count(1) → 3`
**Reverse** - переворачивает список задом наперед: `[1, 2, 3].reverse() → [3, 2, 1]`
**Copy** - создает копию списка: `[1, 2].copy() → [1, 2] (новая копия)`
