# 1. Слова хранятся в виде списка Python прямо в основном файле программы.

```python
words_list = tws.read_words("words.txt")
s = Hangman(words_list)
```

+
- Нет зависимостей от внешних файлов (кроме модуля `txt_words_reader`)
- Быстрая загрузка - данные сразу в памяти
-
- Для добавления новых слов требуется перезапуск программы
- Не подходит для больших словарей

# 2. Слова хранятся в отдельном Python-модуле, который отвечает за чтение данных.

```python
# из файла Hangman.md:
import txt_words_reader as tws

# Из файла txt-reader.py:
def read_words(filename: str):
    words_out = []
    with open(filename, "r", encoding = "utf-8") as myfile:
        words_raw = myfile.readlines()
        for word in words_raw:
            words_out.append(word[:-1])
    return words_out
```

+
- Пере используемость - модуль можно использовать в других проектах
-  Упрощение основного кода - детали чтения файла скрыты
-
-  Дополнительный файл для поддержки
- Нужно следить за совместимостью интерфейса

# 3.  Слова хранятся в простом текстовом файле, по одному слову на строке.

```python
# Использование в коде:
words_list = tws.read_words("words.txt")  # Чтение из файла
s = Hangman(words_list)  # Передача списка в игру
```

+
- Максимальная простота редактирования - можно использовать любой текстовый редактор
- Отделение данных от кода - можно менять слова без изменения программы
-
- Требует проверки существования файла (в текущем коде нет обработки ошибок)
-  Возможны проблемы с кодировкой или форматом
