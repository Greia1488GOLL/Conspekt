## Виртуализация Android 
Отличия Android от ПК-виртуализации:
Архитектура: ARM (Android) vs x86 (ПК) → требуется эмуляция
Производительность: Медленнее, тяжелее для системы
Подход: Эмуляция всего устройства, а не просто ОС
Решения:
Для разработки: Android Studio Emulator
Для игр: BlueStacks, LDPlayer

Для Oracle VM VirtualBox обязательно нужна технология аппаратной виртуализации на процессоре:
Intel: VT-x (Intel Virtualization Technology)
AMD: AMD-V
Без неё виртуальная машина либо не запустится (для 64-битных гостевых ОС), либо будет работать очень медленно в режиме программной эмуляции.
